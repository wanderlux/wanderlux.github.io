## Code

#### Inline

The library you need for basic input and output in c is `stdio.h`. Also, you can compile code by running `gcc mycode.c -o myprogram`

#### Code Blocks

Illustrated with C hello world:

```
    #include <stdio.h>

    void main() {
      printf("Hello world!");
    }
```
